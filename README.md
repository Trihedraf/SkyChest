# SkyChest

SkyBlock Condensed to a Chest

## Changes

### Trading

- Piglins trade Amethyst Shards at the same rate as Nether Quartz

### Crafting

#### Shaped

- 1 Diamond
    - C = 1 Coal/Charcoal
    - G = 1 Gold Ingot
    - I = 1 Iron Block
    - X = Blank
```
GIG
XCX
GIG
```

- 1 Netherite Ingot
    - D = Diamond Block
    - G = Gold Ingot
    - I = Iron Ingot
    - X = Blank
```
GDG
XIX
GDG
```
